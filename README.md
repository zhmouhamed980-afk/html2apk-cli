# html2apk-cli üöÄ

A professional, high-performance CLI tool for Linux designed to convert any web project (HTML/CSS/JS) into a fully functional Android APK using Apache Cordova.

## üåü Key Features

- **Zero Configuration**: Convert simple HTML folders to APK with a single command.
- **Modern Framework Support**: Automatically detects and builds **React, Vite, and TypeScript** projects.
- **Google AI Studio Ready**: Perfect for wrapping AI applications generated by **Google AI Studio** or other AI-prompted web tools into native Android apps.
- **Smart Build System**: Handles `npm install` and `npm run build` automatically, ensuring production-ready assets.
- **Customizable**: Set your App Name, Package ID, Version, and Custom Icon effortlessly.
- **Auto-Fix Paths**: Fixes common "White Screen" issues in Cordova by automatically handling relative paths (`./`) in build configurations.
- **Environment Aware**: Smartly selects compatible Java versions (JDK 17) and detects Android SDK paths.

## üõ† Prerequisites

Ensure you have the following installed on your Linux system:
- **Node.js & npm**
- **Apache Cordova** (`npm install -g cordova`)
- **Android SDK** (Command-line tools)
- **Java JDK 17** (Recommended for Gradle compatibility)

## üìñ Usage

### Basic Usage
Convert a standard HTML folder:
```bash
html2apk-cli ./my-web-folder
```

### Advanced Usage (React/Vite/TS)
The tool will automatically detect `package.json`, run the build process, and package the output:
```bash
html2apk-cli ./my-react-app --name "My Pro App" --id "com.company.myapp" --icon ./icon.png
```

### Options
| Option | Description | Default |
|--------|-------------|---------|
| `source` | Path to your project folder | (Required) |
| `--name` | The name displayed on the Android phone | MyHTMLApp |
| `--id` | Package identifier (e.g., com.example.app) | com.example.htmlapp |
| `--version` | Semantic version string | 1.0.0 |
| `--icon` | Path to a PNG icon | None |
| `--release` | Build as a signed release APK | debug |
| `--output` | Directory for the final APK | dist-apk |
| `--skip-build` | Skip the `npm build` step | False |

## ü§ñ Google AI Studio Projects
If you have a project exported from **Google AI Studio**, simply point `html2apk-cli` to the project folder. The tool will handle the dependencies and build process to give you a ready-to-install APK.
